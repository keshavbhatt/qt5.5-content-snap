# qt5.5-content-snap
qt 5.5 content snap to be consumed by qt 5.5 based apps in snapcraft.io by Keshav Bhatt
# Additional
In addition to qt5.5 runtime this content snap also include the following components:
           - mpv
           - ffmpeg
           - python
           - socat
           - wget
           - coreutils
           - pv
           - libtag1v5
